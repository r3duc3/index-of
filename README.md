# index-of
Download ebook from index-of.es or index-of.co.uk

## Available commands
```
q -> quit                                                                                                                                                                                   
p -> previous                                                                                                                                                                               
n -> next                                                                                                                                                                                   
c -> clear                                                                                                                                                                                  
d -> download                                                                                                                                                                               
s -> set domain                                                                                                                                                                             
h -> help
```

## Installation
```
[sudo] apt install python3 git
[sudo] python3 -m pip install mechanize bs4 requests
git clone https://github.com/r3duc3/index-of
```

## Run it
```
cd index-of
python3 index.py
```

## Requirements
- Python3x
- git
- module python:
  - requests
  - mechanize
  - bs4
